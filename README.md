# Simple Pick-up and Place System

## Environment requirements
- **Godot version:** v4.5.1.stable.official

## Features
- **Object Outline Highlighting**: When the player looks at an object in the scene, its outline will be displayed.
  <center><img width="1600" height="900" alt="Outline Highlighting of the Object" src="https://github.com/user-attachments/assets/acc10d21-5a73-45d4-8262-6463fed30929" /></center>

- **Object Pickup and Place**: Players can pick up objects from the scene and place them anywhere.
  <center><img width="1600" height="900" alt="Pickup and Place Object" src="https://github.com/user-attachments/assets/24485e91-f0de-445a-9c6c-c3edb543f10d" /></center>

- **Blueprint Placement Validation**: After picking up an object, a blueprint determines if the object can be placed. A green highlight indicates a valid placement, while a red highlight indicates an invalid placement.
  
  <center><img width="1600" height="900" alt="Valid Placement" src="https://github.com/user-attachments/assets/e0d288c2-0682-45da-b2ab-38fce32178b7" /></center>
  <center>Valid Placement</center>
  
  <center><img width="1600" height="900" alt="Invalid Placement" src="https://github.com/user-attachments/assets/a02ff884-521f-4346-a1c3-a7f2a2475f12" /></center>
  <center>Invalid Placement</center>

- **Object Rotation**: The picked-up object can be freely rotated with both fast and slow rotation options.

- **Grounded Placement**: Objects are always placed in a way that they stay grounded and don't float above the floor.
  <center><img width="1600" height="900" alt="Object can be placed in the scene" src="https://github.com/user-attachments/assets/566cdb16-2d5f-487a-8358-04810148757c" /></center>

- **Basic Player Movement**: The player can move using basic keyboard inputs.

- **Pause and Resume Functionality**: The game features basic pause and resume capabilities.

- **Adjustable character variables**: Adjust these variables to modify character effects.
